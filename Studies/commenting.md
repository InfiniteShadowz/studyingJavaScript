# Commenting
used usually to communicate the intent of your code, for others or for your future self(trust me, youÂ´ll forget)

## One Line Comment

- **`//`** ----> comment one line only:

_ex:_

```javascript
// This is a comment.
```

## Multi-Line Comment

- **`/*`** -----> starts multi-line comment
- **`*/`** -----> ends multi-line comment
_ex:_

```javascript
/* I am multi-line
commenting */ 
```
